Challenges & Learnings

Dataset Class Imbalance

Common errors drowned out rare failures, biasing the model.

Solved

Implemented synthetic injection and stratified splitting for balance.

Model Cold-Starts

Loading a 110M param model caused 10s+ serverless timeouts.

Solved

Used FastAPI lifespan hooks to preload models into memory on start.

Multi-Lingual Context

"Hinglish" usage confused the standard BERT tokenizer.

Solved

Expanded dataset with colloquial reports to normalize mixed patterns.
